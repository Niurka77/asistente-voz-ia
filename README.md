# 🎙️ Asistente de Voz IA

Un asistente inteligente capaz de procesar comandos de voz y ejecutar acciones automatizadas. Enfocado en la interacción humano-computadora mediante librerías modernas de Python.

### 🛠️ Tecnologías
- **Lenguaje:** Python 3.x
- **IA/ML:** SpeechRecognition, Pyttsx3
- **Lógica:** Automatización de tareas de escritorio

### 🚀 Cómo funciona
El asistente escucha a través del micrófono, procesa el audio para convertirlo en texto y ejecuta la lógica programada respondiendo de forma auditiva.
